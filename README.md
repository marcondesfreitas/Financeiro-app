# Controle Financeiro

Aplicação web desenvolvida em React + TypeScript para gerenciamento de receitas e despesas pessoais.

## Funcionalidades

* Cadastro de receitas e despesas
* Listagem de transações
* Filtro por tipo (Receitas, Despesas e Todas)
* Resumo financeiro

  * Saldo total
  * Total de receitas
  * Total de despesas
* Persistência de dados utilizando LocalStorage
* Interface responsiva
* Desenvolvido com React, TypeScript e Tailwind CSS

## Tecnologias Utilizadas

* React
* TypeScript
* Vite
* Tailwind CSS

## Como Executar

### Instalar dependências

```bash
npm install
```

### Executar o projeto

```bash
npm run dev
```

O projeto estará disponível em:

```txt
http://localhost:5173
```
